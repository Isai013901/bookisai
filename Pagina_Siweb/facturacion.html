Facturación (CFDI Versión 3.3)

A través de este sistema puede emitir las facturas de sus ventas; que contemplan todos los requisitos fiscales requeridos por las autoridades fiscales, incluyendo el CFDI por complemento de pagos, asi como llevar el control de sus cuentas por cobrar.

Sus principales  beneficios:

  1. Contar con una base de sus clientes
  2. Definir un catálogo de los conceptos que factura regularmente
  3. Modificar el sistema para cubrir una necesidad particular
  4. Emitir y visualizar una pre factura, antes de timbrar
  5. Si emite cada mes una factura al mismo cliente y por el mismo importe se puede generar de manera automática cada mes.
